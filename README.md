## Deployment

Download zip file at [here](https://github.com/thamardaw/atd-deploy).

## Lab Deploy

- Change directory into atd-lab-deploy.

- Pull remote images.

```
$ docker-compose pull
```

- Start services.

```
$ docker-compose up -d
```
- Stop services.

```
$ docker-compose down
```
- Web can be accessed at http://localhost.
